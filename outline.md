# Mastering GitHub — 10-Course Coursera Specialization

Progressive ladder from zero to agentic GitHub workflows. Each course ~60 minutes,
3–5 minute videos.

**Target audience:** Developers, DevOps engineers, and technical leaders who want
to master GitHub's platform — from first commit through AI-assisted development to
autonomous agent workflows.

**Structure:** Course → Module (2+ lessons + Critical Thinking Assessment) → Lesson (3–5 videos).
Each lesson produces `key-terms.md` and `reflection.md`. Each module ends with a
**Critical Thinking Assessment** (quiz + `roleplay.md`).
